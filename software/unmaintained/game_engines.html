Project description

Dingux Game Development Kit is a powerful and simple multimedia library for Dingux. It supports the Dingoo A320 and compatible devices.

Gopher Game Development Kit is a powerful and simple multimedia library for Gopher 2.

Retro Game Development Kit is a powerful and simple multimedia library for portable game consoles with RetroFW firmware.
